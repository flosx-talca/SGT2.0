Familias de Evaluación (Motores Genéricos)
Las familias definen la lógica base (comparación, rangos, calendarios). Los hijos (reglas) heredan esta lógica y aplican parámetros específicos.
Código Familia Descripción de Lógica Función Genérica Estado Acciones
comparison Compara valores con umbrales (==, !=, >, <) eval_comparison Core
range Valida pertenencia a rangos (min/max) eval_range Core
sequence Valida secuencias de días o patrones eval_sequence Core
calendar Evalúa domingos, feriados y ausencias eval_calendar Core
