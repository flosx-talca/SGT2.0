# README.txt

## Cuál archivo usar

### Si trabajas en Cursor
Usa estos archivos como contexto principal:
- .cursor/rules/sgt-context.mdc
- .cursor/rules/scheduling.mdc
- PROJECT_CONTEXT.md

Cursor entiende mejor las reglas específicas por archivos y contexto del proyecto.

### Si trabajas con Antigravity u otro agente compatible con AGENTS.md
Usa:
- AGENTS.md
- PROJECT_CONTEXT.md

AGENTS.md está pensado como una guía general para agentes de código.

## Recomendación
Mantén siempre actualizado PROJECT_CONTEXT.md.
Luego sincroniza las reglas específicas en Cursor o AGENTS.md según la herramienta que uses.

## Orden de lectura sugerido
1. PROJECT_CONTEXT.md
2. especificacion-funcional-sistema-turnos.md
3. Archivo específico de la herramienta que estés usando
